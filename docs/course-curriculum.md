# 강의 설계안 — "Claude Code로 만드는 법원경매 투자분석 자동화"

이 문서는 `auction_harness`로 구축한 경매 투자분석 파이프라인을 **따라 하며 배우는 실전형 온라인 강의**로 재구성한 커리큘럼이다.

## 1. 강의 개요

| 항목 | 내용 |
|------|------|
| 과정명(안) | AI 에이전트로 완성하는 부동산 경매 투자분석 자동화 |
| 부제 | 물건 발굴 → 시세·권리분석 → 투자판단까지, 나만의 분석 하네스 구축 |
| 대상 | 경매 투자자 / 부동산 데이터 분석 실무자 / AI 자동화 입문~중급 |
| 레벨 | 입문~중급 (비개발자도 따라올 수 있게, CLI·Python은 강의 내 보충) |
| 선수지식 | 기본 PC 활용. 경매 기초 용어·공공데이터 API는 강의에서 제공 |
| 결과물 | 관심 지역 물건을 자동 수집·분석해 투자등급 리포트를 생성하는 파이프라인 |
| 형식 | VOD(녹화강의) + 실습 핸즈온 + 캡스톤 프로젝트 |

### 학습 성과 (수료 후)
1. Claude Code 하네스(에이전트+스킬) 설계·구축
2. 법원경매 물건 자동 수집·지역/용도/유찰 필터링
3. 공공데이터 API(실거래가·공동주택)로 시세·단지 스펙 분석
4. 매각물건명세서 기반 권리분석(대항력·인수금액·명도) 자동화
5. 발굴→분석→랭킹→리포트 end-to-end 파이프라인 운영

## 3. 총 소요시간 요약

| 구분 | 영상(VOD) | 실습 | 합계 |
|------|:--:|:--:|:--:|
| Module 0~6 (필수) | 약 6시간 30분 | 약 4시간 15분 | 약 10시간 45분 |
| Module 7 (심화) | 50분 | 20분 | 1시간 10분 |
| Capstone | – | 2시간 | 2시간 |
| 전체 | 약 7.3시간 | 약 6.6시간 | 약 14시간 |

- 순수 VOD 분량: 약 7~8시간 (30~40개 차시, 차시당 10~35분)
- 실습 포함 실제 수강시간: 약 14~16시간 (개인차·API 승인 대기 제외)
- 권장 수강 페이스: 주 2~3시간 × 5~6주 완주

## 4. 제작·운영 노트

- 차시 길이: 10~20분 단위로 잘게(집중도·재수강 편의). 실습은 별도 "따라하기" 영상 분리.
- 부교재: 각 모듈 실습 스크립트·템플릿·체크리스트 제공(복붙 실행 가능).
- 환경 편차 대응: OS별(mac/Win) 세팅 보충 영상, API 미승인 시 대체 데이터셋.
- 법적 고지 필수: "투자 참고자료이며 판단·손실 책임은 투자자 본인" 명시 + 스크래핑 이용약관·개인정보(임차인명 등) 취급 주의(모듈 7-3).
- 차별화 포인트: 수기 분석 대신 AI로 수십 건을 자동 분석하는 것이 핵심 셀링.
