# 경매 투자 리포트 템플릿

`05_report.html` 생성용 상세 가이드. 톤은 `algo-design`(크림 배경 + 오렌지 액센트)을 따른다.

## 색상 (등급)
| 등급 | 색 | 의미 |
|------|-----|------|
| S | `#E8590C` (진오렌지) | 고수익·저리스크 |
| A | `#F08C00` | 우수 |
| B | `#748FFC` (블루) | 보통 |
| C | `#ADB5BD` | 주의 |
| D | `#868E96` (회색) | 비추천 |

## 섹션 순서
1. 헤더 — 분석 조건(지역·물건종류·기간), 생성일시.
2. 요약 카드 행 — 총 물건수 / 분석 완료 / 미분석 / 등급 분포.
3. 랭킹 표 — 정렬: 종합점수 내림차순. 컬럼: 순위·사건번호·소재지·등급 뱃지·기본ROI·리스크점수·한줄근거.
4. 상위 물건 상세 카드(S·A만) — 3시나리오 ROI 막대, 총투자금 분해(낙찰가+인수금액+취득비용+이자), 예상낙찰가 vs 실거래 중앙값.
5. ⚠️ 주의 물건 섹션 — 리스크플래그 2개↑ / 인수과다 / data_gap.
6. 면책 문구.

## 원칙
- 표는 `overflow-x: auto` 컨테이너로 감싸 모바일에서 가로 스크롤.
- 라이트/다크 모두 대응(`prefers-color-scheme`).
- 숫자는 천단위 콤마, ROI는 % 표기, 금액은 만원 단위.
- 데이터가 없는 셀은 빈칸이 아니라 `—`로 표기.
